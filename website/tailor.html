Tailor scrolls and filters a ReadSeeker line by line and writes the data into a Writer.

func New

func (*Tailor) Start

func (s *Tailor) Start() chan error

Start is the goroutine for reading, filtering and writing. The returned chan emit an error in case of failure, otherwise it's closed upon tail completion without emitting nothing.

func (*Tailor) Stop

func (s *Tailor) Stop()

Stop will causes the tail operation to ends. The log file is readed until EOF before stopping.
