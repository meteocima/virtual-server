# virtual-server

`virtual-server` contains a set of functions
that simplify the orchestration of long running
tasks.

* Copy
* Exec
* Exists
* Link
* MkDir
* Move
* ReadDir
* ReadString
* RmDir
* RmFile
* Run
* WriteString
